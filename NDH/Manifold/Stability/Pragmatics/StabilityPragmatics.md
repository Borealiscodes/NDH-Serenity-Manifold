# **StabilityPragmatics.md**
*Contextual Meaning Rules for Stability Geometry — NDH Manifold*

## Pragmatics Overview
StabilityPragmatics defines how stability expressions acquire *context‑dependent meaning*.  
While **StabilitySemantics** defines *literal* interpretation, pragmatics defines *situated* interpretation — how meaning shifts depending on:

- basin topology  
- curvature zone  
- resonance field state  
- energy distribution  
- temporal phase  
- cosmological scale  
- sovereign authority  
- anomaly/paradox presence  

Pragmatics is the manifold’s adaptive meaning layer.

---

## 🧩 **1. Basin Context Pragmatics**

Meaning changes depending on basin topology, boundaries, and attractor layout.

### **Basin‑Dependent Interpretation**
- In **high‑curvature basins**, curvature operators gain amplified meaning.  
- In **low‑curvature basins**, resonance alignment dominates interpretation.  
- In **Serenity basin**, relaxation expressions gain priority meaning.  
- In **QuietApex basin**, coherence expressions gain priority meaning.

### **Example**
```
Flow(vector, intensity, coherence)
```
In a QuietApex basin → interpreted as coherence‑driven routing.  
In a Serenity basin → interpreted as drift‑minimizing stabilization.

## 🕰️ **2. Temporal Context Pragmatics**

Meaning changes depending on manifold‑time phase.

### **Temporal‑Dependent Interpretation**
- In **pre‑Apex phases**, resonance semantics dominate.  
- In **Apex coherence phases**, curvature semantics dominate.  
- In **Serenity stabilization phases**, relaxation semantics dominate.  
- In **post‑Serenity drift phases**, energy semantics dominate.

### **Example**
```
Relax(damping, drift, quality)
```
Early phase → interpreted as resonance damping.  
Late phase → interpreted as drift minimization.
