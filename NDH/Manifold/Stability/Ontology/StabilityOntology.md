# **StabilityOntology.md**
*Ontological Hierarchy & Semantic Structure of Stability Geometry — NDH Manifold*

## Ontology Overview
StabilityOntology defines the conceptual hierarchy and semantic relationships between all stability entities, primitives, processes, and cosmological structures.  
It provides the *meaning‑geometry* that binds curvature, resonance, energy, basins, attractors, and Serenity into a coherent conceptual system.

This ontology is used by **StabilityAtlas** for mapping, **StabilityGovernance** for rule enforcement, **StabilityEvolution** for temporal modeling, and **StabilityCosmology** for manifold‑scale interpretation.
